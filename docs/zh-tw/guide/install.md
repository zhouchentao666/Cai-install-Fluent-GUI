# 安裝 FluentInstall

## 1. 下載程式
二進位檔案：
| 來源 | 連結 |
| --- | --- |
| GitHub | [點擊下載](https://github.com/zhouchentao666/Fluent-Install/releases/latest) |
| 國內雲盤 | [點擊下載](https://wwazq.lanzoub.com/b00odsy2kd)（密碼: 1234） |
| 開源地址 | [GitHub](https://github.com/zhouchentao666/Fluent-Install/) |

- 或者從上方開源地址下載最新版本的releases或原始碼

## 2. 安裝依賴

- 確保已經安裝了 Python 3.11 或更高版本
- 執行 `fluent_install.bat` 安裝必要的依賴
- 等待安裝完成

## 3. 啟動程式

- 執行 `python main.py` 啟動程式
- 或者執行 `run_fluent.bat` 快速啟動

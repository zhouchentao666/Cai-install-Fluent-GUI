# Frequently Asked Questions

## 1: Software shows successful addition but game not in library

Please check the following:

0. Make sure you have used the repair tool. If not, go to [3a.lol](https://3a.lol) to download it.

1. Is SteamTools installed and is unlock mode enabled?

2. Is Steam running a beta version? If so, disable it.

3. Check if SteamTools has "Disable game updates" enabled. If so, disable it.

4. Check for conflicts with other library tools (e.g., GreenLuma, scam tools, fake CDKs, etc.). Remove all if present. If still not working, reinstall Steam completely.

5. If none of the above works, reinstall SteamTools.

6. If still not working, reinstall Steam.

7. If none of the above methods work, the SteamTools server may be down. Please wait.

---

## 2: Cannot Open Steam

**Solution**: In SteamTools, set Steam to **Online Mode**, then start Steam

## 4: Common Errors and Solutions

### No Internet Connection

- Check network connection
- lua/manifest/error
- Poor network, try multiple times
- Clear cache
- **Uncheck DLCs that have no size**

### Content Still Encrypted

- **Cause**: lua file expired
- **Solution**: Update lua file
- **Uncheck DLCs that have no size**

### Content Not Available

- **Cause**: Manifest/lua error or expired
- **Solution**: Update manifest or lua

### Download 0KB

- **Cause**: Invalid or incorrect lua/manifest
- **Solution**: Re-obtain valid lua or manifest

### Invalid Installation Path

- **Cause**: Missing token in lua
- **Solution**: Check lua configuration

### 6-.../54-... Error

- **Cause**: Missing ticket in lua
- **Solution**: Update lua file

### No License

- **Cause**: Unlocker (SteamTools, etc.) not working properly
- **Solution**: Refer to "Game not in library after successful addition" solution

---

## 💡 Tips

If you encounter the above issues and your game has DLC, try disabling some DLCs to see if it downloads. If still not working, find another library source.

For unresolved issues, please visit communities (such as caigamer.cn) to find game manifests.

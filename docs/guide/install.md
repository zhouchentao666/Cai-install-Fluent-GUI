二进制文件：
| 来源 | 链接 |

| GitHub | [点击下载](https://github.com/zhouchentao666/Fluent-Install/releases/latest) |

- 或者从上方开源地址下载最新版本的releases或源代码

## 2. 安装依赖

- 确保已经安装了 Python 3.11 或更高版本
- 运行 `fluent_install.bat` 安装必要的依赖
- 等待安装完成

## 3. 启动程序

- 运行 `python main.py` 启动程序
- 或者运行 `run_fluent.bat` 快速启动

# 使用程序

## 1. 搜索并入库游戏

1. 点击左侧导航栏的**"搜索入库"**
2. 在搜索框中输入游戏名称或 AppID
3. 点击**"搜索"**按钮
4. 从搜索结果中选择你要入库的游戏
5. 选择**清单源**（如果某个源失败，请尝试其他源）
6. 根据需要勾选入库选项：
   - **添加所有 DLC**：同时入库游戏的所有 DLC
   - **修补 Depot Key**：入库游戏的创意工坊内容
   - **修补 Manifest**（不建议选，不能用再选）：修复清单文件
7. 点击**"入库游戏"**按钮

## 2. 管理已入库游戏

1. 点击左侧导航栏的**"主页"**
2. 查看所有已入库的游戏列表
3. 可以切换视图模式（列表/卡片）和排序方式
4. 点击游戏卡片上的**"删除"**按钮可以移除已入库的游戏

## 3. 重启 Steam

1. 入库完成后，点击右上角的**"重启 Steam"**按钮
2. 确认重启后，Steam 将自动关闭并重新启动
3. 重启后，入库的游戏将出现在你的库中

## 4. 设置

1. 点击左侧导航栏的**"设置"**
2. 可以配置以下选项：
   - **Steam 路径**：手动指定 Steam 安装路径（留空则自动检测）
   - **主题模式**：浅色/深色/跟随系统
   - **主题颜色**：选择你喜欢的主题颜色
   - **语言**：切换界面语言
   - **默认页面**：设置启动后显示的默认页面
   - **解锁工具模式**：自动检测/强制 SteamTools/强制 GreenLuma
   - **入库超时时间**：网络较慢时可适当增大

## 5. 注意事项

1. **确保 SteamTools 已安装并正常运行**
2. **入库前建议先重启 SteamTools**
3. **如果入库失败，尝试更换清单源**
4. **某些游戏可能需要特定的清单源才能入库成功**
5. **入库后必须重启 Steam 才能看到游戏**
